# Agent Banking Suite

Open-source platform for agent banking, POS operations, merchant management, and digital payments.

## Features

- Agent Management
- Merchant Management
- Wallet Services
- QR Payments
- Transaction Processing
- Analytics Dashboard

## Technology Stack

### Mobile
- Flutter

### Backend
- Node.js
- Express
- PostgreSQL

### Infrastructure
- Docker
- GitHub Actions

## Project Structure

This repository (`agent-sdk-flutter`) contains the core Flutter SDK for the Agent Banking Suite. It provides easy-to-use helpers and classes for:
- Wallet generation and management
- QR Payment processing
- Transaction handling

Other planned repositories in the AgentLabs ecosystem include:
- `agent-core-api`
- `agent-mobile`
- `agent-admin`

## Roadmap

- Authentication Integrations
- Wallet Management Helpers
- POS Operations Support
- QR Payments Utilities
- Reporting and Analytics Tools
- Blockchain Integrations (e.g., Stellar)
