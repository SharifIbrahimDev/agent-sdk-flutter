# Contributing to AgentLabs

Thank you for your interest in contributing to the Agent Banking Suite! We welcome contributions from the community.

## Workflow

1. **Fork the repository**
2. **Create a branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/my-feature
   ```
3. **Commit your changes** using descriptive commit messages:
   ```bash
   git commit -m "feat: add wallet generator"
   ```
4. **Push your changes** to your fork:
   ```bash
   git push origin feature/my-feature
   ```
5. **Create a Pull Request** against the `main` branch of this repository.

## Requirements

- **Follow coding standards**: Ensure your Dart code complies with our `analysis_options.yaml` rules. Run `flutter analyze` before submitting a PR.
- **Add tests**: Please write unit tests for any new functionality you add. Run `flutter test` to ensure all tests pass.
- **Update documentation**: If your changes affect the API or user behavior, please update the relevant documentation in `README.md` and inline comments.

## Good First Issues

If you're looking for a place to start, check out the issues labeled `good first issue` or `help wanted` in our issue tracker!
